Navigation

Creating the navigation was not as bad as I first thought it would be. The HTML was what we learnt in class with the 5 pages needed within the website. The CSS however consisted of a few things such as the background colour (thanks to coolors I ended using a palette consisting of nice colours throughout the webpages), padding, grid area since we needed grid formatting and then some hover features and actually being clickable.

Main Page

The main page was a normal page which used the grid layout we were taught in class to basically introduce what the website is about, which in this case was it being a portfolio about myself. The grid layout consisted of a header, navigation menu, aside, main and a footer at the end. The HTML and CSS for these were kinda easy since it was colours and spacing them out evenly. With the grid layout used they had to be columned out which I used percentages and they needed their own areas to be situated in. I also messed around with some animations when trying to implement some. This allowed me to implement keyframes with transformation properties to move images/gifs around in the screen.

Project Page

The project page was the page which mainly had to include the projects which makes this website as a portfolio, however since I have not done any individual projects yet I have used filler images that will hopefully be replaced when I start making and coding my own projects in the future which I am looking forward to. It can use some spacing and a couple more pictures but I think it will look better when actual projects get added within it.

Contact Page

This page consisted of an email form using the mailto service which prompts an email service when filling in the fields and pressing the button to send a message. I will admit it was quite difficult using css to create the box to be in the center of the page with its contents neatly inside, however apart from it being an email form it ended up being okay and not too over the top. It also allowed me to write extra comments at the bottom to add some words to the page.
